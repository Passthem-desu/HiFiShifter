# 1. 介绍
HiFiShifter 是一款开源免费的人力Vocaloid 制作以及人声修音软件，支持在同一套工程中完成拼字、调音与导出。软件界面和操作习惯参考了 Reaper 和 VocalShifter ，整体逻辑是先处理时间线上的切片，再在参数面板里做细调。

# 2. 拼字
## 2.1 音频导入
HiFiShifter 常用导入方式有：
1. 从系统文件管理器拖拽音频到轨道区域。
2. 从 HiFiShifter 的文件管理器拖拽到轨道。
3. 在轨道区域按 `Ctrl + F` 打开快捷搜索导入框。

导入建议：
- 长音频先做切片再进入调音流程，会更流畅。

## 2.2 轨道管理与切片编辑
HiFiShifter 支持拖拽轨道排序和父子轨结构。处于根一级的轨道与其子轨组成一个轨道组，共享一套参数面板。实际使用中，建议按“一个声部一个轨道组”去整理。

轨道面板基本操作：
- 滚轮：水平缩放
- Ctrl + 滚轮：垂直缩放
- 中键拖拽：平移时间线

切片常用编辑：
- `S`：在播放头位置切分选中切片
- 拖拽切片边界：调整范围
- 拖拽切片角点：调整淡入淡出
- `Ctrl + C / Ctrl + V`：复制粘贴切片
- `Alt + 左键`拖拽切片边界：拉伸缩短音频
- 按住`Shift`拖动切片：取消吸附

轨道上有一个关键开关 `C (compose)`。只有开启 `C` 的根轨，才会进入后续调音流程。

# 3. 调音
点击根轨 `C` 后，HiFiShifter 会对该轨道组进行分析，请等待任务完成后再开始编辑。分析时间会受到切片数量和音频长度影响。

## 3.1 参数面板基本操作
参数面板基本缩放移动逻辑为：滚轮水平缩放，Ctrl + 滚轮垂直缩放，中键拖拽平移。

常用编辑方式：
- 左键绘制参数曲线
- 右键恢复到原始曲线
- Select 模式下支持选区复制粘贴曲线（`Ctrl + C / Ctrl + V`）
- 可以按tab切换编辑、选区、直线绘制模式
- 工具栏有许多小功能按钮，可以自行尝试

## 3.2 音高面板使用建议
音高面板会显示每个 clip 的参考信息

选中 clip 后，可使用 `-` / `=` 对该时间范围内的参数线做整体上下移动。音高按半音移动，便于快速做整体转调。

# 4. 工程管理
HiFiShifter 工程文件后缀为 `hshp`。

HiFiShifter 支持直接打开 Reaper 和 VocalShifter 工程，但工程较大时可能占用较多资源，建议先做简化再导入。

# 5. 导出
在左上角菜单选择 `文件-导出` 进行导出，可选择总体导出或分轨导出。

# 6. 常用快捷键速查
均可以在 `视图-快捷键管理` 中修改。

| 操作                         | 快捷键 / 鼠标                   |
| :--------------------------- | :------------------------------ |
| 平移视图（时间轴）           | 鼠标中键拖动                    |
| 横向缩放（时间轴）           | 鼠标滚轮（以光标为中心）        |
| 纵向缩放（轨道高度，时间轴） | Ctrl + 鼠标滚轮                 |
| 纵向缩放（参数轴，参数面板） | Ctrl + 鼠标滚轮（参数面板内）   |
| 播放/暂停                    | Space（空格）                   |
| 播放/停止                    | Enter                           |
| 撤销/重做                    | Ctrl + Z / Ctrl + Y             |
| 新建工程                     | Ctrl + N                        |
| 打开工程                     | Ctrl + Shift + O                |
| 保存                         | Ctrl + S                        |
| 另存为                       | Ctrl + Shift + S                |
| 导出音频                     | Ctrl + E                        |
| 模式切换（选择/绘制）         | Tab                             |
| 删除选中剪辑                 | Delete / Backspace              |
| 复制选中剪辑（应用内剪贴板） | Ctrl + C                        |
| 粘贴到播放头位置             | Ctrl + V                        |
| 参数面板复制选区曲线         | Ctrl + C（Select 模式）         |
| 参数面板粘贴到选区起点       | Ctrl + V（Select 模式）         |
| 分割剪辑                     | S（在播放头位置分割选中的剪辑） |
| 新建轨道                     | Ctrl + T                        |
| 快速搜索                     | Ctrl + F                        |
